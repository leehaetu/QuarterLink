# HMRC Wording and Scope

## Use

| Wording | Use |
| --- | --- |
| Making Tax Digital for Income Tax | Product and compliance context |
| send quarterly updates | User action wording |
| quarterly updates | Update type |
| bridging software | Product category |
| software that connects to records | Plain-language description |
| spreadsheet records | Source record wording |

## Avoid

| Wording | Reason |
| --- | --- |
| MTD ITSA | Avoid shorthand in user-facing text |
| HMRC-approved | Do not claim recognition or approval |
| file quarterly tax returns | Quarterly updates are not tax returns |
| submit quarterly returns | Quarterly updates are not quarterly returns |
| EOPS | Avoid shorthand |
| End of Period Statement | Out of bootstrap and MVP scope |
| VAT features | Excluded from QuarterLink |
| bookkeeping software | QuarterLink is bridging-only software |

## Scope Rule

QuarterLink is bridging-only software for Making Tax Digital for Income Tax. Anything not specified here should be written as:
"To be specified in a later ticket."
